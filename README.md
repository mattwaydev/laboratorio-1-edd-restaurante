# 🍽️ Restaurante La Merlina

Sistema de gestión para restaurante desarrollado en Java con interfaz gráfica Swing y archivos secuenciales.

## 👥 Equipo
| Nombre | Módulo |
|--------|--------|
| Mateo Carrascal | CRUD de Platos, Persistencia y Lógica |
| Samuel Ramírez | CRUD de Mesas, Persistencia y Lógica |
| Sebastian Anaya | Interfaz gráfica de todas las pantallas |

## ⚙️ Funcionalidades
- ✅ CRUD de Platos (crear, editar, eliminar, cambiar disponibilidad, buscar)
- ✅ CRUD de Mesas (crear, editar, eliminar, cambiar estado)
- ✅ Crear y gestionar pedidos
- ✅ Agregar platos a un pedido con validación de disponibilidad
- ✅ Consultar pedido por ID
- ✅ Cerrar pedido y generar factura automáticamente
- ✅ Cancelar pedido
- ✅ Reportes de ventas y operación
- ✅ Persistencia total en archivos `.txt`

## 🛠️ Tecnologías
- Java JDK 23 — NetBeans 25 — Swing — FlatLaf

## 🚀 Cómo ejecutar
1. Clona el repo en NetBeans
2. Asegúrate de tener la carpeta `data/` con los `.txt`
3. En `src/ui/` ejecuta `MenuPrincipal.java` con Run
